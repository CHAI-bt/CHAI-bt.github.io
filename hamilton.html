🔄 勒让德变换与相空间

从拉格朗日表述 $(q, \dot{q})$ 到哈密顿表述 $(q, p)$ 的数学桥梁：

$$H(q, p, t) = \dot{q}_i p_i - L(q, \dot{q}, t)$$

其中 $p_i \equiv \partial L / \partial \dot{q}_i$ 为共轭动量。

相空间：由 $n$ 个广义坐标和 $n$ 个共轭动量构成的 $2n$ 维空间，系统的任一状态对应一个点，演化对应一条曲线。相空间是位形空间的余切丛。

💡 勒让德变换的条件：$L$ 关于 $\dot{q}$ 是凸函数，以保证 $p$ 与 $\dot{q}$ 一一映射。变换可逆，$f(x)$ 和 $g(p)$ 等价。

📐 哈密顿正则方程 考试重点

$$\dot{q}_i = \frac{\partial H}{\partial p_i}, \qquad \dot{p}_i = -\frac{\partial H}{\partial q_i}$$

$2n$ 个一阶方程取代 $n$ 个二阶拉格朗日方程。$q_i$ 和 $p_i$ 的地位近乎对称。

哈密顿量守恒：若 $H$ 不显含时间，则 $dH/dt = \partial H/\partial t = 0$，$H$ 守恒（但不一定等于总能量）。

简谐振子示例：
$L = \frac12 m\dot{x}^2 - \frac12 kx^2$, $p = m\dot{x}$
$H = \frac{p^2}{2m} + \frac12 kx^2$
$\dot{x} = p/m$, $\dot{p} = -kx$。
循环坐标：
若 $q_j$ 不在 $H$ 中出现，则 $\dot{p}_j = -\partial H/\partial q_j = 0$，$p_j$ 守恒。

⚡ 电磁场中的带电粒子

拉格朗日量：$L = \frac12 m v^2 - q\phi + q\mathbf{A}\cdot\mathbf{v}$，对应广义势 $U = q\phi - q\mathbf{A}\cdot\mathbf{v}$。

正则动量：$\mathbf{p} = m\mathbf{v} + q\mathbf{A}$ （不是机械动量！）

哈密顿量：

$$H(\mathbf{x}, \mathbf{p}) = \frac{(\mathbf{p} - q\mathbf{A})^2}{2m} + q\phi$$

这里 $H$ 等于总能量，但 $H \neq T + V$，因为粒子的势能只来源于电势 $\phi$，而磁矢势 $\mathbf{A}$ 贡献了动量中的 $q\mathbf{A}$ 项。

🧲 潘宁阱（Penning Trap）

由旋转双曲面电极（罩电极、环电极）和轴向磁场 $B$ 构成，用于约束带电粒子。

电势：$\Phi = \frac{U_0}{2z_0^2 + r_0^2}(2z^2 - x^2 - y^2)$

在柱坐标下求得哈密顿量，导出三个特征运动频率：

  • 轴向简谐振动频率 $\omega_z$
  • 修正回旋频率 $\omega_+$ 和磁控管频率 $\omega_-$

条件 $\omega_c^2 - 2\omega_z^2 > 0$ 确保粒子被约束。1989年诺贝尔奖相关工作即利用潘宁阱精密测量电子g因子。

🔄 3D转动：欧拉角与四元数

刚体转动由正交矩阵 $A$ ($A^T A = I$, $\det A = +1$) 描述，属于 $\mathrm{SO}(3)$ 群。共9个参数，3个独立。

欧拉角 $(\phi, \theta, \psi)$：通过三次转动 $A = BCD$ 得到。

角速度在体坐标系下的分量：

$$\boldsymbol{\omega} = \begin{pmatrix} \dot{\phi}\sin\psi\sin\theta + \dot{\theta}\cos\psi \\ \dot{\phi}\cos\psi\sin\theta - \dot{\theta}\sin\psi \\ \dot{\phi}\cos\theta + \dot{\psi} \end{pmatrix}$$
⚠️ 万向节锁死：当 $\theta=0$ 时，$\phi$ 和 $\psi$ 不再独立，损失一个自由度。欧拉角的参数空间是三维环面 $T^3$，而转动群是 $\mathrm{SO}(3)$（拓扑为 $\mathbb{RP}^3$），无法全局覆盖。

四元数 / SU(2) 表示：凯莱-克莱因参数 $Q \in \mathrm{SU}(2)$，参数空间为三维球面 $S^3$，双覆盖 $\mathrm{SO}(3)$，避免锁死。对应关系：$Q = q_0 I + i\mathbf{q}\cdot\boldsymbol{\sigma}$，$\sum q_i^2 = 1$。

🪀 重陀螺的定点转动

对称陀螺 $I_1 = I_2 \neq I_3$，拉格朗日量用欧拉角表示为：

$$L = \frac{I_1}{2}(\dot{\theta}^2 + \dot{\phi}^2\sin^2\theta) + \frac{I_3}{2}(\dot{\phi}\cos\theta + \dot{\psi})^2 - Mgl\cos\theta$$

$\phi$ 和 $\psi$ 是循环坐标，对应动量 $p_\phi$、$p_\psi$ 守恒。能量守恒给出等效一维运动：

$$\frac{I_1}{2}\dot{\theta}^2 + V_{\text{eff}}(\theta) = \text{const}$$

规则进动（无章动，$\dot{\theta}=0$）满足：$Mgl = \dot{\phi}(I_3\omega_3 - I_1\dot{\phi}\cos\theta_0)$。

分为慢进动和快进动，慢进动频率 $\dot{\phi} \approx Mgl/(I_3\omega_3)$。

拓展至磁场中的磁偶极子：拉摩进动 $\omega_L = -\gamma B$。

🎯 莫佩蒂原理（最小作用量原理 Δ 变分）

在 等能不等时 的变分下，真实路径使简约作用量取极值：

$$\Delta \int_{t_1}^{t_2} \sum_i p_i \dot{q}_i\, dt = \Delta \int_C p_i\, dq_i = 0$$

当约束和势场均与时间和速度无关时，雅可比形式将其化为位形空间的最短路径问题：

$$\Delta \int \sqrt{2(E-V)}\, ds = 0$$

示例：均匀重力场中质点轨迹，通过该原理可直接解出抛物线。

🔗 与哈密顿原理的区别：哈密顿原理（δ变分）是等时不等能；莫佩蒂原理（Δ变分）是等能不等时。
