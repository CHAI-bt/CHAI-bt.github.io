Topic 6: Naming and Necessity（命名与必然性）

本节回答：一个名字到底是「贴在对象上的标签」还是「一串描述的缩写」？克里普克（Saul Kripke）在 1970 年代发动了一场革命——他证明日常名字不是描述，科学发现也可以是「必然的」。整章围绕一条主线：先理解罗素/弗雷格的老看法（kc6.1 背景），再学克里普克的核心武器「严格指示词」（kc6.2），然后用它推出两个惊人结论——后天必然真理（kc6.3）和先验偶然真理（kc6.4），最后看克里普克怎么用它为「本质主义」辩护（kc6.5）。

先修背景：名字的旧图景——罗素与弗雷格

在克里普克之前，哲学家对「名字怎么指东西」有两种主流回答。先看清它们，才能体会克里普克的革命性。

罗素（Russell, 1918）主张描述理论：日常名字（如「苏格拉底」）其实是缩略描述——「柏拉图的老师、饮鸩而死的哲学家」。名字通过描述的中介才接触到对象。

弗雷格（Frege, 1892）区分了Sinn（sense / 涵义）与Bedeutung（reference / 指称）。晨星和暮星指同一颗行星（指称相同），但给出方式不同（涵义不同）——这解释了为什么「晨星 = 暮星」是有信息量的发现，而不是废话。

为什么需要这段背景：它们都假设名字要「勾住」对象，必须走描述这条路。克里普克要炸掉的，恰恰是这条路。

背景

核心枢纽：严格指示词——克里普克的革命

先看一个直觉：鲁迅 = 周树人。这个等式是真的。但——如果周树人小时候没被送去读书，他还会是「鲁迅」吗？直觉上会——「鲁迅」这个名字跟着这个人走，不管他有什么性质。

从这个直觉抽出克里普克的核心概念：严格指示词（Rigid Designator）——在每一个可能世界（possible world）里，只要那个对象存在，它就指那个对象；在对象不存在的可能世界里，它什么也不指。

「可能世界」不是科幻——只是「事情本来可以不同的方式」的严谨说法。比如「鲁迅没有弃医从文」就是一个可能世界（历史上没发生，但逻辑上可以想象）。在那个世界里，「鲁迅」仍然指同一个人。

为什么需要这个概念：它是克里普克整个理论的发动机。一旦你接受「名字是严格指示词」，后面关于「必然」和「先验」的所有结论都会跟着跑。

易错：「严格」不是「严肃」，是「固定不变」——指称在所有可能世界保持同一个。别和「严格」的日常用法混淆。

必考

严格指示词的对比表：谁严格，谁不严格？

表达式是否严格指示词？为什么？
专名（鲁迅、北京、孔子）✅ 是命名仪式 + 因果链固定了指称，不随对象性质变化而变。
自然种类词（水、金、老虎）✅ 是（克里普克的主张）「水」指 H₂O，在所有可能世界里只要 H₂O 存在就指 H₂O——即使那里的人管它叫「XYZ」。
某些限定描述（「第 79 号元素的原子」）✅ 是描述锁定了一个本质属性（原子序数），跨可能世界不变。
典型描述（「那位写《呐喊》的人」）❌ 不是鲁迅可能没写《呐喊」——在那个世界里「那位写《呐喊》的人」就指别人了。
摹状词（「现任法国国王」）❌ 不是不同可能世界里的「现任法国国王」可以是不同的人。

怎么用这张表：题目问一个词是不是严格指示词，你就问——「在某个可能世界里，这个词会不会指一个不同的对象？」如果会，它就不是严格指示词。

高频

名字的因果链理论：克里普克怎么替代描述理论

如果名字不是通过「描述」指称对象，那它是怎么做到的？克里普克的回答：因果链（causal chain）。

想象一个场景（来自 Kripke 1980, p. 91）：一个婴儿出生了，父母指着他说「我们叫他 Richard Feynman」。这个名字通过一条链传来传去——朋友告诉同事、同事告诉学生、学生告诉记者……传到几千里外一个从未见过 Feynman 的人嘴里，他仍然指着 Feynman 用这个名字。

关键：这个名字传到你耳朵里的时候，不需要你知道任何关于 Feynman 的描述。你只要在因果链的末端，用这个名字去指那个起点的那个人就行了。

为什么需要它是对描述理论的替代：描述理论要求你知道对象的特征才能指称它。因果链理论说——不需要。你只需要「站在这条链上」就够了。这就解释了为什么一个对 Feynman 一无所知的普通人也能成功地用「Feynman」指称他。

必考

后天必然真理：科学发现可以是「必然的」

传统上，哲学家默认一个二分法——

  • 先验（a priori）= 不需要经验就能知道 → 如「2+2=4」「单身汉是未婚的」
  • 后验 / 经验（a posteriori）= 需要观察才知道 → 如「北京在下雨」「水在 100°C 沸腾」
  • 必然（necessary）= 不可能为假 → 如「2+2=4」「H₂O 有两个氢原子」
  • 偶然（contingent）= 可能为假 → 如「北京在下雨」「鲁迅写了《呐喊》」

传统配对是：先验 = 必然（理性主义），后验 = 偶然（经验主义）。克里普克打破了这个配对。

例子：「晨星 = 暮星」（Morning Star = Evening Star）。

  • 这是后验的——你必须看天空才能知道，光靠想是想不出来的。
  • 但它是必然的——晨星和暮星其实是金星（Venus）。「金星 = 金星」是必然的。

另一个例子：「水 = H₂O」。这是化学发现的，显然是后验的。但克里普克说——水是严格指示词，H₂O 也是。两个严格指示词如果在一个可能世界指同一个对象，就在所有可能世界都指同一个对象。所以「水 = H₂O」是必然的。

为什么需要它：它把「先验/后验」和「必然/偶然」这两对概念拆开。前者是认识论概念（我们怎么知道），后者是形而上学概念（事情本身是否可能不同）。混在一起是旧图景的错误。

易错：「水 = H₂O」的必然性不是说「我们不需要经验就能知道」，而是说「一旦通过经验发现了，它就是跨可能世界都成立的事实」。「后天必然」≠「不需要经验」，而是「经验发现的 + 不可能为假」。

必考

先验偶然真理：反过来也成立

克里普克还指出另一类传统上被认为不可能的组合：先验偶然真理。

例子（来自 Kripke）：标准米尺 S 在时间 t₀ 的长度。

  • 我们定义「1 米 = 标准米尺 S 在 t₀ 的长度」。这是先验的——不需要测量就能知道。
  • 但它是偶然的——S 在 t₀ 可能因为受热膨胀而变长。在那个可能世界里，「1 米」就不是 S 的长度了。

这个例子展示了「先验」和「偶然」的组合也是可能的——先验来自定义方式，偶然来自定义项本身是可变的。

为什么需要它和后天必然形成对称对比：两对概念独立配对，四种组合都可能。这是克里普克对哲学地图的一次重要修正。

重点

先验/后验 × 必然/偶然：克里普克的完整地图

必然（跨所有可能世界为真）偶然（在某些可能世界为假）
先验（不靠经验就知道）「2+2=4」「三角形有三条边」「标准米尺 S 的长度 = 1 米」
后验（靠经验才知道）「水 = H₂O」「晨星 = 暮星」「北京在下雨」「鲁迅写了《呐喊》」

怎么用这张表：题目给一个命题，先判断「需要先验经验还是经验观察才能知道」（先验/后验），再判断「它是否可能为假」（必然/偶然），然后定位到对应格。

必考

本质主义：克里普克为「本质」正名

从严格指示词 + 后天必然，克里普克推出一个惊人的哲学立场：本质主义（Essentialism）——事物有某些跨可能世界都成立的「本质属性」。

论证（水的例子）：

  • 「水」是严格指示词 → 在所有可能世界里，水都指 H₂O。
  • 假设在某个可能世界里，那里有一种无色透明可喝的液体，化学结构是 XYZ，人们管它叫「水」。
  • 克里普克的判断：那不是水。那只是「假水」——因为水 = H₂O 是必然的。

这等于说：H₂O 是水的本质——没有 H₂O，就不是水，不管它表面上多像水。

金（gold）的例子同理——原子序数 79 是金的金（本质属性），金块的颜色、光泽只是偶然属性。

为什么需要它：它复兴了亚里士多德时代的「本质主义」，但在分析哲学的严格框架里重建。它告诉科学——科学发现的不是偶然规律，而是必然的形而上学事实。

必考

本质属性 vs. 偶然属性：对比表

对象本质属性（跨可能世界不变）偶然属性（在某些可能世界不同）
水化学结构 = H₂O温度、状态（冰/液/气）、所在行星
金原子序数 = 79颜色（可能因杂质变化）、形状、市场价值
鲁迅与周树人是同一个人（= 因果链起点）是否弃医从文、是否写了《呐喊》
北京地理位置（华北平原那个城市）人口、GDP、是否首都

怎么用这张表：题目问「X 是否可能是非 Y」，你就问「Y 是 X 的本质属性吗？」如果 Y 是本质属性，那 X 不可能是非 Y；如果不是，那在某些可能世界里 X 可以是非 Y。
