🔗 约束与广义坐标

完整约束：可表示为 $f(r_1, r_2, \dots, t)=0$，减少独立变量数。

非完整约束：不能写成上述形式，可能涉及不等式或不可积微分约束。

$N$ 个质点有 $3N$ 个坐标，$k$ 个完整约束后，自由度为 $3N-k$，引入 $n = 3N-k$ 个广义坐标 $q_1, \dots, q_n$。

💡 核心思想：广义坐标提供了一种自然的处理受力系统的方法。位形空间是 $n$ 维微分流形，体系演化对应其中的一条曲线。

⚡ 达朗伯原理

理想约束下（约束力虚功为零 $\sum_i f_i \cdot \delta r_i = 0$），外力和倒转有效力所做虚功之和为零：

$$\sum_i \left(F_i^{(a)} - \dot{p}_i\right) \cdot \delta r_i = 0$$

通过将虚位移 $\delta r_i$ 转为广义坐标变分 $\delta q_j$，引入广义力 $Q_j \equiv \sum_i F_i \cdot \frac{\partial r_i}{\partial q_j}$。

$Q_j$ 的量纲不一定是力，但 $Q_j \cdot q_j$ 总是功的量纲。

🎯 哈密顿原理 考试重点

单演系统在通过位形空间中两个固定点的所有可能运动中，真实路径使作用量取极值：

$$\delta I = \delta \int_{t_1}^{t_2} L(q, \dot{q}, t)\, dt = 0$$

由此导出拉格朗日方程。此即最小作用量原理的"δ变分"形式：等时不等能。

📐 拉格朗日方程

$$\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}_j}\right) - \frac{\partial L}{\partial q_j} = 0$$

$L = T - V$ 为拉格朗日量（标量），适用于完整约束、单演系统。

🔹 规范不变性：
$L' = L + \frac{dF(q,t)}{dt}$ 给出相同的运动方程。
🔹 广义势推广：
若 $F_i = -\frac{\partial U}{\partial r_i} + \frac{d}{dt}\frac{\partial U}{\partial \dot{r}_i}$，拉格朗日方程仍成立，$L = T - U$。

示例：电磁场中带电粒子 $L = \frac{1}{2}mv^2 - q\phi + q\mathbf{A}\cdot\mathbf{v}$，其中 $U = q\phi - q\mathbf{A}\cdot\mathbf{v}$ 为速度依赖的广义势。

🔧 非完整约束与拉格朗日乘子法

对于 Pfaflian 形式的约束 $\sum_k a_{\alpha k} dq_k + a_{\alpha t} dt = 0$，引入乘子 $\lambda_\alpha$：

$$\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}_j}\right) - \frac{\partial L}{\partial q_j} = \sum_\alpha \lambda_\alpha a_{\alpha j} = Q_j$$

$Q_j$ 即为广义约束力。这同样适用于完整约束，可求出约束力。

⚠️ 示例：无滑滚下斜面的铁环，约束 $R\dot{\theta} = \dot{x}$ 可写为 $f(\theta, x) = R\theta - x = 0$，乘子法解得静摩擦力 $F_c = \frac{mg\sin\phi}{2}$。

⭐ 对称性与诺特定理 考试重点

诺特定理：作用量的每个连续对称性对应一个守恒量。

$$C_m = \sum_j \frac{\partial L}{\partial \dot{q}_j} Q_j = \text{const}$$
空间平移不变性
→ 总动量守恒 $\mathbf{P} = \sum_i \mathbf{p}_i$
空间旋转不变性
→ 总角动量守恒 $\mathbf{L} = \sum_i \mathbf{r}_i \times \mathbf{p}_i$
时间平移不变性 → 能量函数 $h = \sum_j \dot{q}_j \frac{\partial L}{\partial \dot{q}_j} - L$ 守恒。若 $\frac{\partial L}{\partial t}=0$ 且约束与时间无关、势场与速度无关，则 $h = E = T+V$。

广义动量：$p_j \equiv \partial L / \partial \dot{q}_j$，若 $q_j$ 为循环坐标，则 $p_j$ 守恒。

🪐 有心力问题

两体问题通过约化质量 $\mu = m_1 m_2/(m_1+m_2)$ 转化为等效单体有心力问题。利用角动量守恒 $l = mr^2\dot{\theta} = \text{const}$，轨道方程（比奈方程）：

$$\frac{d^2u}{d\theta^2} + u + \frac{m}{l^2}\frac{dV(1/u)}{du} = 0, \quad u \equiv \frac{1}{r}$$

平方反比力 $V = -k/r$ 给出圆锥曲线 $1/r = C(1+e\cos\theta)$：$e<1$ 椭圆，$e=1$ 抛物线，$e>1$ 双曲线。

📌 伯特兰定理：所有被束缚质点中，只有平方反比力和胡克定律（$f \propto r$）能给出闭合轨道。

🌍 受限三体问题与拉格朗日点

$m \ll m_1, m_2$，两主天体做圆周运动。在同步旋转非惯性系中，有效势：

$$V_{\text{eff}} = -\frac{1}{2}\Omega^2(x^2+y^2) - \frac{Gm_1}{r_1} - \frac{Gm_2}{r_2}$$

拉格朗日点满足 $\nabla V_{\text{eff}} = 0$，共 5 个：

  • L1, L2, L3：位于两主天体连线上（共线点），不稳定（鞍点）。
  • L4, L5：与两主天体构成等边三角形，在 $m_2/m_1 \lesssim 0.04$ 时稳定。

L1、L2 与较小天体的距离为希尔半径 $r_H = R\left(\frac{m_2}{3m_1}\right)^{1/3}$。
